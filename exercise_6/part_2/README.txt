
I. File List
------------
Makefile
src/main.c
src/ApproximationOfPi.c
src/ApproximationOfPi.h

II. Compilation
---------------
In order to compile, use the Makefile:

$ make ApproximationOfPi

To delete the compiled files:

$ make clean

III. Usage
----------
Run the heat relaxation: 

$ ./ApproximationOfPi <NUM_ITERATIONS>

NUM_ITERATIONS	How many iterations should be made.
